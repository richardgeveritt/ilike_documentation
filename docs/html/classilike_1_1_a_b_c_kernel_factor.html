Detailed Description

An ABCKernel type of Factor for use in ExactLikelihoodEstimator.

The ABCKernelFactor class is used in ExactLikelihoodEstimator, which contains LikelihoodFactors, DistributionFactors, and IndependentProposalKernels. These are evaluated when a Particle is evaluating its factors in a weight update or Metropolis step, for example.
