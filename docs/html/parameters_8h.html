Detailed Description

This file contains the declaration and implementation of the Parameters class.

The Parameters class is designed to manage and manipulate parameters, which can be either Armadillo matrices or any other type using Boost's any type. The class supports various operations such as deep copying, merging, and accessing parameters by name.

Dependencies:

  • RcppArmadillo
  • Boost (unordered_map and any)
